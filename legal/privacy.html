1. Ruang Lingkup

Dokumen ini menjelaskan bagaimana situs Verb 1-2-3 Finder (Aether Studio) menangani data saat Anda menggunakan fitur pencarian kata kerja.

  • Aplikasi berbasis client-side: pemrosesan terjadi di browser Anda.
  • Kami tidak membuat akun, tidak meminta login, dan tidak mengirim data pencarian ke server.

2. Data yang Diproses

Secara default, situs ini tidak mengumpulkan data pribadi. Data yang mungkin diproses di perangkat Anda:

  • Kata kunci pencarian: hanya untuk menyaring daftar kata kerja di halaman ini.
  • Preferensi tampilan (mis. jumlah item per halaman): disimpan lokal di browser (opsional) agar pengalaman konsisten.

Data tersebut tidak dikirim ke server dan tidak dibagikan ke pihak mana pun.

3. Cara Penggunaan Data

Semua data hanya digunakan untuk fungsi inti situs (pencarian/filter/penyalinan ke clipboard). Tidak ada profil pengguna, pelacakan perilaku, atau iklan bertarget.

4. Penyimpanan & Keamanan

  • Jika diaktifkan, preferensi disimpan sebagai localStorage di perangkat Anda.
  • Anda dapat menghapusnya kapan saja melalui pengaturan browser (“Clear Site Data”).
  • Kami menganjurkan selalu mengakses situs lewat protokol aman (HTTPS) saat dipublikasikan.

5. Cookies & Pihak Ketiga

  • Cookies: tidak digunakan secara default.
  • Analitik: tidak aktif. Jika suatu saat ditambahkan, kami akan memperbarui kebijakan ini dan menampilkan banner persetujuan.
  • Embed pihak ketiga (mis. ikon, CDN): hanya untuk aset statis. Tidak digunakan untuk pelacakan.

6. Hak Pengguna

Karena kami tidak menyimpan data pribadi di server, permintaan akses/hapus data biasanya tidak diperlukan. Namun Anda selalu bisa:

  • Menghapus data lokal (cache, localStorage, cookies) di browser.
  • Menghubungi kami jika ada pertanyaan terkait privasi.

7. Privasi Anak

Situs ini dapat dipakai pelajar. Kami tidak mengumpulkan data dari anak di bawah usia 13 tahun.

8. Perubahan Kebijakan

Kami dapat memperbarui kebijakan ini sewaktu-waktu untuk menyesuaikan fitur/teknologi baru. Tanggal “Terakhir diperbarui” akan diupdate.
